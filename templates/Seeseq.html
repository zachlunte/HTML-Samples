======== Seeseq ========

DESCRIPTION:
A tool for visualizing number sequences.
Currently only supports a handful of visualization schemes.

Future versions will:
 - Include more visualization options
 - Include more number sequence choices
 - Improve calculation and display time efficiency
 - Add new features to the user interface
IMAGES:
DOWNLOAD INSTRUCTIONS:
1. Click on "Seeseq-v1.tar.gz" in the "Download Links" section.

2. Save the file.

3. Open the Terminal application located at /Applications/Utilities/Terminal.app

4. Navigate to the Downloads folder by entering the following command:

$ cd ~/Downloads/

5. Un-zip and un-tar the tar bundle using the following command:

$ tar zxvf Seeseq-v1.tar.gz

6. A new Seeseq folder should now exist in the ~/Downloads/ directory.

7. Navigate into this new Seeseq folder using Finder.

8. Double-click the Seeseq.jar file.

9. For more information, read the README.txt file in the Seeseq folder.
