CURRENT PROJECTS

======== dieTunes ========

DESCRIPTION:
A simple Java-based music player for macOS.
Currently only supports WAV, AIFF, AU audio formats.

Future versions will:
 - Extend support to other file formats
 - Add new features to the user interface
IMAGES:
DOWNLOAD INSTRUCTIONS:
1. Click on "dieTunes-v1.tar.gz" in the "Download Links" section.

2. Save the file.

3. Open the Terminal application located at /Applications/Utilities/Terminal.app

4. Navigate to the Downloads folder by entering the following command:

$ cd ~/Downloads/

5. Un-zip and un-tar the tar bundle using the following command:

$ tar zxvf dieTunes-v1.tar.gz

6. A new dieTunes-v1 folder should now exist in the ~/Downloads/ directory.

7. Navigate into this new dieTunes-v1 folder using Finder.

8. Double-click the dieTunes-v1.jar file.

9. For more information, read the README.txt file in the dieTunes-v1 folder.
DOWNLOAD LINKS:
>: dieTunes-v1.tar.gz
